About this app

      
    
        
  • Chromium only. Uses the Web Serial API (Chrome / Edge / Opera, desktop).
    • 
        
  • USB, not WiFi. This app talks to the receiver over the USB serial port. It never uses WiFi/HTTP, so mixed-content blocking does not apply.
    • 
        
  • Offline & installable. A service worker caches the app shell, so it loads without a network. Use your browser's Install app option to add it to your desktop.
    • 
        
  • Setup. On the receiver enable Settings → USB Port → Ad hoc, plug in USB, then click Connect (115200 8N1).
    • 
        
  • Telemetry (frequency, band, mode, volume, RSSI, SNR, battery) is read from the device status stream. RDS and per-point spectrum data are not exposed over this serial protocol, so they are not shown.
