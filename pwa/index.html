About this app

      
    
        
  • Chromium only. Uses the Web Serial API (Chrome / Edge / Opera, desktop).
    • 
        
  • USB, not WiFi. Talks to the receiver over the USB serial port (115200 8N1). It never uses WiFi/HTTP.
    • 
        
  • Offline & installable. A service worker caches the app shell; use your browser's Install app option.
    • 
        
  • Setup. On the receiver enable Settings → USB Port → Ad hoc, plug in USB, then Connect.
    • 
        
  • Full parity with the device web UI via the extended ? serial protocol (status, scan, memory, config).
